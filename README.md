1. HTML Basics
2. CSS Basics
3. Intermediate CSS
4. Box Model
5. Text, Images and Textbox
6. Organization
7. Inline and Block elements
8. Containers
9. Nesting Techniques
10. Grid, Flexbox and Nesting Flexbox
11. Positioning
12. Media Queries
